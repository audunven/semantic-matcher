1-1.) I can't download WordNet.

Three causes typically account for most instances of this problem:

  • You're trying to sign in as root.
  • Your browser isn't configured to send your email address to the FTP server.
  • You're trying to download from behind a firewall.

2-1.) I've downloaded the Windows package but don't know how to install it.

You should have downloaded a file called "WordNet-1.7.1.exe". If you downloaded it correctly, you should be able to simply double-click on this file and it should extract itself. This step simply unbundles the package. Then you must follow the instructions in the INSTALL.txt file to actually install the WordNet package.

3-1.) [Windows] I've installed WordNet successfully, but it says it can't find the database files.

If you haven't already, reboot your computer and try running it again; the Windows version of WordNet needs to modify your autoexec.bat file before it will run correctly.

3-2.) What is the number to the right of the sense, in parentheses, in the overview of senses?

This number designates the number of times the sense is tagged.
